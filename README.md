# CSV-Demo
CSV-Demo is application used to upload file and store data into database. Application built using JAVA/J2EE, Spring Boot, Hibernate, JPA. 
UI is designed with HTML, CSS, Bootstrap, jQuery and AngularJS.

Application configuration is placed under src/main/resources
application.properties update according to system configuration.

Import database using db.sql.

Import code in IDE and Sping Boot application can be run from Application.java, it has embeded tomcat server.

Generate WAR from maven install command and deploy in external application server.
